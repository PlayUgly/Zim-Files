====== Pacman ======
Created Saturday 20 August 2016

**sudo pacman -Syu**		updates Arch
**sudo pacman -S [Package Name]	**installs package
**pacman -Ss [Package Name]**	searches for package
**pacman -Q**			list of installed packages
